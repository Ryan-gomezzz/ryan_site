# Ryan Gomez — Portfolio

Next.js (App Router) portfolio site.

## Deploy on Vercel

1. Push this repository to GitHub.
2. In [Vercel](https://vercel.com/new), import the repo **Ryan-gomezzz/ryan_site**.
3. Use the defaults: **Framework Preset: Next.js**, **Root Directory: `.`**, **Build Command:** `next build`, **Output:** Next.js (handled automatically).

No extra environment variables are required for the current build.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
